Fuente de alimentación.

Fuente de alimentación regulable de 0 a 36 Voltios y 10 a 20 Amperios utilizando un LM317. El proyecto se divide en un esquema TOP dónde divide el circuito en cuatro partes.

Adaptación de la Fuente. Circuito fuente y fuentes extras. Circuito fuente para alimentar ventilador y multímetro. Multimetro, ventiladores, resistencia shunt.
